# Md. Fahim Mahmud - Portfolio

Welcome to the source code for my professional portfolio website!

## Overview
This is a lightweight, responsive, completely static web application built using:
- **HTML5** & **Vanilla JavaScript** (Zero build-step architecture)
- **Tailwind CSS v4** (Utility-first styling via native CDN)
- Native `IntersectionObserver` APIs for scroll-based motion triggers

## Deployment
Because this project utilizes a build-less architecture, it is designed to be hosted directly on GitHub Pages. No Node.js, Vite, or npm compilation steps are required. Simply serving the `index.html` file will run the entire application.
